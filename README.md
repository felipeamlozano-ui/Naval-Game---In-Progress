Jogo Naval — HTML, CSS e JavaScript

Este projeto é uma versão web do clássico jogo Batalha Naval, desenvolvida com HTML, CSS e JavaScript. A proposta é aplicar conceitos de estruturação de páginas, estilização com Grid e lógica de programação para criar uma experiência interativa no navegador.

O tabuleiro possui dimensão 7x7, tornando a partida mais dinâmica e objetiva.

Objetivo

Desenvolver um jogo funcional no qual o jogador pode:

Selecionar posições no tabuleiro para atacar

Receber feedback visual de acertos e erros

Afundar todos os navios para vencer a partida

Tecnologias Utilizadas

HTML5 — Estrutura da aplicação

CSS3 — Layout e estilização com CSS Grid

JavaScript (ES6+) — Lógica do jogo e manipulação do DOM

Funcionalidades

Geração dinâmica de um tabuleiro 7x7

Posicionamento automático de navios

Sistema de clique para realizar ataques

Atualização visual imediata após cada jogada

Verificação automática de condição de vitória

Bloqueio de jogadas repetidas na mesma célula
